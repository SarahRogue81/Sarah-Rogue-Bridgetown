== 🐧 my linux love & terminal tweaking

i am a huge gnu/linux enthusiast. i love customized setups and digging deep into config files. here are some of my personal tech highlights:

*   **the fish shell**: i absolutely love the command shell `fish`even though i'm a complete noob when it comes to fish (and actual fishing as well), i was determined to get my terminal looking and acting exactly how i wanted. when i switched from *CachyOS* to **Fedora 42**, i spent hours racking my tiny brain to replicate CachyOS's epic prompt, custom aliases, and virtual environments. 
*   **kde plasma & google drive hacks**: when dolphin (the kde file manager) refused to play nice with google drive on arch linux, i didn't give up. i hacked the `google.provider` xml file using kate, replaced it with gnome's provider configuration, logged in with my passkeys, and got it working beautifully!
*   **the great winboat experiment**: i tried out *WinBoat* to run windows 11 pro inside docker on gnu/linux. i completely messed up the initial installation and had a nightmare of a time uninstalling the docker images. once i got it running via qemu in my browser, i realized something funny—i didn't even need windows! my gaming framerates on cachyos are identical to my native windows partition, and since i can run microsoft 365 in my browser, i can happily say "sorry, microsoft"
